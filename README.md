# Setup Guide for Python Autotests
 
## Python Setup
1. Create virtual environment (in project directory):
   ```bash
   python3 -m venv .venv
   ```
 
2. Activate virtual environment:
   ```bash
   # On Windows:
   .venv\Scripts\activate
   ```
 
3. Install requirements:
   ```bash
   pip install -r requirements.txt
   ```
